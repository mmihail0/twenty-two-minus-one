# round resolving logic

Option 2 (un-stand) — player takes their full turn first, then enemy acts one card at a time. While the enemy is going, SHIFT lets you un-stand and take another hit. TAB stays usable throughout so you can still play ability cards reactively.

Option 3 (alternating) — player hits or stands, then the enemy automatically responds with one action after a 1 second delay, then control returns to the player. Once both have stood the round resolves. The enemy only uses its ability card on its very first action of the round.

it's called option 2 and option 3 because im testing both of them for a permament one. option 1 was so bad it was removed but the 2/3 convention stayed.
